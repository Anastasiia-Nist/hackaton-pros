### Setup

```BASH
git clone https://github.com/sergasent/hackaton-pros.git
cd hackaton-pros/
npm ci
```

#### Run in production mode

```BASH
npm run build
npm run start
```

Then open `http://localhost:4173/` in browser

#### Run in dev mode

```BASH
npm run dev
```

Then open `http://localhost:5174/` in browser
